# Deep Packet Inspection IDS using CNN algorithm

## Overview
This project is an IDS system using CNN to detect malicious packets.

## Features
- Packet inspection
- Dashboard display
- Telegram alerts

## Tools Used
- Virtualbox (Ubuntu)
- Python
- VSCode
- CTU-13 Dataset
